The Blood Donor App

“If you’re a blood donor, you’re a hero to someone, somewhere, who received your gracious gift of life.”

  • Simple User Interface
  • Find donors/seekers faster
  • Hospitals & Blood banks numbers
Start Exploring
E-mail Authentication

User have to sign up with the credentials and after registering they can sign in. Firebase Authentication is used here.

Nearby Donors

Users can search for donors nearby and can see the nearby seekers. They can also make direct phone call to them.

Offline Support

Firebase offline support keeps the loaded data available even if user is not connected to the internet.

Hospitals

User can see list of all the hospitals in their area and can also see the location and make a call.

Blood Banks

User can see list of all the blood banks in their area and can also see the location and make a call.

Request Blood

User can post a request for blood and can get a response from a bunch of available donors.

Find Donor/Seeker Nearby!

Using this app you can find a perfect donor or seeker for you. This feature will suggest you the nearest seeker for you if you want to donate blood.

Also you can find a donor near to you. More features are,

  • Large Database
  • Can update profile
  • Automatically reminds you to donate
